# Auctionary Full-Stack Project

This is a full-stack auction application consisting of a Node.js backend and a Vue.js frontend.

## Project Structure
- **/backend**: Express.js API and Database logic.
- **/frontend**: Vue.js user interface.

## How to Run
1. **Backend**: Navigate to /backend, run \
npm install\, then \
Node server.js\.
2. **Frontend**: Navigate to /frontend, run \
Npm install\, then \
Npm run dev\.
